#Project Pawrkour Notes

Project Pawkour is a Unity-based parkour game where the player controls a cat trying to escape from a lab.

Core movement (running, jumping, wallrunning)
First level complete with an end-of-level scene
Fully animated cat model
High fidelity user interface 
Dynamic music response 

problems encountered: 
Animation Consistency and Blending
Wallrunning State Verification
Neon Material Fragment Shader
Camera Clipping
Unity Version Control Limitations

Music is reactive to gameplay. music changes on the speed of the cat.
